## Username Permutator
It can convert people's full names into hundreds of potential valid usernames.

> John Doe => 140 Usernames  
> Johnny Doeboy => 428 Usernames  
> Longjohnathan Superdoeman123 => 2180 usernames

### Requirements
- Python 3
- Click
>pip install click
